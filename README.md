# Watson

Browser-based intelligence platform for rapid data capture and visualization.

**Version:** BETA | **Version:** 1.0 | [Try it](https://watson.notalex.sh/)

---

## Quick Start

1. Type in the editor to capture notes
2. `Shift + Tab` to add entities
3. Link entities as you create them
4. Save with `Ctrl/Cmd + S`

---

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Quick Add | `Shift + Tab` |

| Save Project | `Ctrl/Cmd + S` |
| Open Project | `Ctrl/Cmd + O` |

| Map View | `Ctrl/Cmd + 4` |

---

## Editor Commands

| Command | Result |
|---------|--------|
| `/` | Quick insert menu |
| `/now ` | Insert current date/time |
| `@name` | Mention entity |
| `#event` | Mention event |
